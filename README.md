# Quiz Website

A static quiz website built with HTML, CSS, and JavaScript. This project includes user authentication pages, interactive quiz functionality, a leaderboard, and supporting informational pages.

## Project Overview

The Quiz Website is designed to help users test their knowledge across different subjects while tracking progress and leaderboard standings.

## Key Features

- Home page with featured quizzes and user dashboard sections
- Authentication pages for login, registration, and logout
- Interactive quiz flow with subject selection, timed questions, and result feedback
- Leaderboard display for ranked user performance
- About Us and Contact pages for project information
- Responsive navigation and page layout

## Folder Structure

- `index.html` - Main landing page
- `auth/` - Authentication pages
  - `login.html`
  - `logout.html`
  - `register.html`
- `css/` - Stylesheets for page layouts and components
- `images/` - Project images and icons
- `js/` - JavaScript logic for authentication, quiz flow, navigation, and results
- `pages/` - Secondary pages for quiz, leaderboard, profile, result, about us, and contact

## Getting Started

### Open the Project

1. Open the project folder in your code editor.
2. Open `index.html` in your browser.

### Recommended Workflow

- Start at `index.html` to access the main site.
- Use the `Quiz` page to take quizzes after logging in.
- Visit the `Leaderboard` page to view rankings.
- Use `About Us` and `Contact` for additional information.

## Notes

- This project is a static website; no backend server is required to run it.
- For the best experience, host the files on a local server or open them directly in a modern browser.
